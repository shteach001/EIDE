置日志信息 方法   操作系统支持：Windows    所属对象：互联网筛选器

修改日志信息。修改后的日志信息，将替换原有日志信息被IIS写入日志中。仅当“取事件类型()”返回值为 #筛选器事件.记录日志 的情况下才能调用本方法。

语法：  逻辑型  互联网筛选器．置日志信息 （［主机名］， ［用户名］， ［服务器名］， ［操作类型］， ［目标］， ［参数］， ［HTTP状态码］， ［Win32错误码］， ［发送字节数］， ［接收字节数］， ［处理时间(ms)］）

参数名 描 述
主机名 可选的； 文本型。The client's host name。
用户名 可选的； 文本型。The client's user name。
服务器名 可选的； 文本型。The name of the server to which the client is connected。
操作类型 可选的； 文本型。The HTTP operation type。
目标 可选的； 文本型。The target of the HTTP command。
参数 可选的； 文本型。The parameters passed to the HTTP command。
HTTP状态码 可选的； 整数型。The HTTP return status。
Win32错误码 可选的； 整数型。The Win32 error code。
发送字节数 可选的； 整数型。The number of bytes sent from the server to the client。
接收字节数 可选的； 整数型。The number of bytes received by the server from the client。
处理时间(ms) 可选的； 整数型。单位毫秒。The time in milliseconds required to process the client request。
