筛选器拒绝理由 枚举常量集合类型   操作系统支持：Windows    跳至：互联网服务支持库

参见：互联网筛选器.取拒绝访问信息() 第三个参数。

成员 描 述
登录 常量值为 1 。The client could not be logged on. SF_DENIED_LOGON。
访问控制列表 常量值为 2 。A Windows DACL denied the resource. SF_DENIED_RESOURCE。
筛选器 常量值为 4 。An ISAPI filter denied the request. SF_DENIED_FILTER。
扩展程序或CGI 常量值为 8 。An ISAPI extension or CGI application denied the request. SF_DENIED_APPLICATION。
服务器配置 常量值为 65536 。The server configuration denied the request. For example, disabling anonymous requests on the server would generate this filter notification when a user without credentials tried to make a request to the server. SF_DENIED_BY_CONFIG。
