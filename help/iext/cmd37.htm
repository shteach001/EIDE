插入按钮 方法   操作系统支持：Windows    所属对象：工具条

将指定按钮插入到工具条中，返回插入后该按钮所处的位置。

语法：  整数型  工具条．插入按钮 （［插入位置］， ［类型］， ［图片索引］， ［标题］， ［动态提示文本］， ［按钮状态］）

参数名 描 述
插入位置 可选的； 整数型。指定新按钮插入的位置索引，0 为栏目一，1 为栏目二，如此类推。如果提供 -1 ，则插入到工具条的尾部。如果本参数被省略，默认值为 -1 。
类型 可选的； 整数型。按钮的类型可以为以下常量值之一： 0: #普通按钮； 1: #多选按钮； 2: #单选按钮； 3: #下拉按钮； 4: #分隔条。 如果本参数被省略，默认值为“#普通按钮”。
图片索引 可选的； 整数型。图片索引用于指定图片组属性中的某张图片，该图片将被显示在按钮表面。索引值从 0 开始，-1 表示无图片。如果本参数被省略，默认值为 -1 。
标题 可选的； 文本型。本参数指定显示在按钮下的标题文本。如果被省略，默认值为空文本。
动态提示文本 可选的； 文本型。本参数指定当鼠标移动到按钮上并停留一段时间后所动态显示的提示文本。如果被省略，默认值为空文本。
按钮状态 可选的； 整数型。本参数指定按钮的存在状态。状态值为 0 或者以下常量值任意组合(相加)： 1: #选中； 2: #按下； 4: #禁止； 8: #隐藏； 16: #变灰。 其中“选中”状态在类型为多选按钮或单选按钮时有效，“按下”状态在类型为普通按钮时有效。如果本参数被省略，默认值为 0 。
