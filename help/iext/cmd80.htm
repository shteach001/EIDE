插入列 方法   操作系统支持：Windows    所属对象：超级列表框

在报表型列表框中指定位置插入新的列，成功返回新列的位置索引，失败返回 -1 。

语法：  整数型  超级列表框．插入列 （［插入位置］， ［标题］， ［列宽］， ［对齐方式］， ［图片索引］， ［图片是否居右］）

参数名 描 述
插入位置 可选的； 整数型。指定新列插入时的位置索引，索引值从 0 开始。如果提供 -1 ，则插入到尾部。如果本参数被省略，默认值为 -1 。
标题 可选的； 文本型。本参数指定报表列的标题文本。如果被省略，默认值为空文本。
列宽 可选的； 整数型。本参数指定报表列的宽度，单位为像素。如果被省略，默认值为 50 。
对齐方式 可选的； 整数型。参数可以为以下常量之一： 0：#左对齐； 1：#中间对齐； 2：#右对齐。如果本参数被省略，默认值为“#左对齐”。
图片索引 可选的； 整数型。图片索引用于指定“表头图片组”属性中的某张图片。索引值从 0 开始，-1 表示无图片。如果本参数被省略，默认值为 -1 。
图片是否居右 可选的； 逻辑型。指定图片是否显示在右边，如果本参数被省略，默认值为假。
