getAttributeStringCollection 方法   操作系统支持：Windows    所属对象：IWMPMediaCollection

Returns the string collection associated with an attribute。

语法：  IWMPStringCollection  IWMPMediaCollection．getAttributeStringCollection （bstrAttribute， bstrMediaType）

参数名 描 述
bstrAttribute 必需的； 文本型。
bstrMediaType 必需的； 文本型。
