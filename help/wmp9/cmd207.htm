getMediaAtom 方法   操作系统支持：Windows    所属对象：IWMPMediaCollection

Gets an atom associated with an item name which can be requested from an IWMPMedia out of this collection via getItemInfoByAtom。

语法：  整数型  IWMPMediaCollection．getMediaAtom （bstrItemName）

参数名 描 述
bstrItemName 必需的； 文本型。
