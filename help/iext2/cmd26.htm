提取加入图片 命令   操作系统支持：Windows    所属类别：图片组处理

从指定文件中提取其中的某一图标，并将其加入到图片组中。成功返回所加入图标在被加入图片组中的位置索引，该索引值从 0 开始。如果失败，返回 -1 。

语法：  整数型  提取加入图片 （图片组数据变量， 图标所在文件， ［图标索引］， ［提取小图标］）

参数名 描 述
图片组数据变量 必需的； 字节集，参数数据只能提供变量。本参数提供即将被处理的图片组数据，处理后的图片组数据也被存放回此变量中。
图标所在文件 必需的； 文本型。本参数提供图标资源所在的文件名(譬如EXE、DLL或ICO等文件)。
图标索引 可选的； 整数型。本参数提供被提取图标在指定文件内所有图标中的索引位置，从 0 开始。如果本参数被省略，默认值为 0 。
提取小图标 可选的； 逻辑型。如果本参数为真，将从图标所在文件中提取小图标（通常为16*16大小）；如果为假，则提供大图标（通常为32*32大小）。如果本参数为真但没有相应小图标，自动提取相同索引处的大图标。如果本参数被省略，默认为假。
