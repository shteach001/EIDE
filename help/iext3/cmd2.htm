加入项目 方法   操作系统支持：Windows    所属对象：卷帘式菜单

在指定菜单中加入一个新的项目。调用本方法前，指定的菜单必须已经存在。

语法：  逻辑型  卷帘式菜单．加入项目 （项目名称， ［菜单索引］， ［项目索引］， ［图片索引］， ［项目数值］）

参数名 描 述
项目名称 必需的； 文本型。
菜单索引 可选的； 整数型。指定在哪个菜单中插入项目。索引从 0 开始。如果本参数省略，表示在当前菜单中插入项目。
项目索引 可选的； 整数型。指定项目插入的位置。索引从 0 开始。如果本参数省略，表示在指定菜单（前一个参数）的最后插入项目。
图片索引 可选的； 整数型。指定该项目所使用的图片。索引从 0 开始。如果本参数省略，默认为-1，表示不使用图片。
项目数值 可选的； 整数型。指定与该项目相关联的附加数值，供用户自己使用。如果本参数省略，默认为0。
