平面 普通类型   操作系统支持：Windows    跳至：DirectX3D支持库

成员 描 述
a 小数型； 初始值为 0 。平面方程ax + by + cz + dw = 0的x系数。
b 小数型； 初始值为 0 。平面方程ax + by + cz + dw = 0的y系数。
c 小数型； 初始值为 0 。平面方程ax + by + cz + dw = 0的z系数。
d 小数型； 初始值为 0 。平面方程ax + by + cz + dw = 0的w系数。

方法：

创建平面 创建平面扩展 求点乘四维向量 求点乘三维向量
求正规点乘三维向量 求直线平面交点
