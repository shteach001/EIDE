打开SQL数据库 方法   操作系统支持：Windows    所属对象：外部数据库

本命令是针对 SQL SERVER 数据库的特定打开方法，通过自建 ODBC 连接文本来打开 SQL SERVER 数据库。调用本命令后无需再调用“打开”命令。成功返回真，失败返回假。

语法：  逻辑型  外部数据库．打开SQL数据库 （服务器名称， ［用户名］， ［密码］， ［数据库名称］， ［是否只读］， ［不显示ODBC连接对话框］）

参数名 描 述
服务器名称 必需的； 文本型。本参数提供 SQL SERVER 服务器的名称或其客户端别名。
用户名 可选的； 文本型。本参数提供登录服务器所需要的用户名，如果省略本参数，默认为不需要。
密码 可选的； 文本型。本参数提供登录服务器所需要的密码，如果省略本参数，默认为不需要。
数据库名称 可选的； 文本型。本参数提供即将打开数据库的名称。如果省略本参数，则使用该用户的默认数据库。
是否只读 可选的； 逻辑型。指定是否以只读方式打开。如果参数被省略，默认值为假。
不显示ODBC连接对话框 可选的； 逻辑型。指定当数据源连接失败的时候，是否自动显示ODBC的数据源连接对话框。如果参数被省略，默认值为假。
