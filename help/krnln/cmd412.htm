编辑 命令   操作系统支持：Windows    所属类别：数据库

提供一个对话框，用作查看或编辑修改当前数据库的记录，起始位置从当前记录开始。本命令退出后当前记录指针被移动到首记录。

语法：  无返回值  编辑 （［标题］， ［编辑方式］， ［字段范围］， ... ）

参数名 描 述
标题 可选的； 文本型。本参数指定对话框的标题，如果被省略，默认为“数据库记录编辑”。
编辑方式 可选的； 整数型。可以为 0 或者以下常量值的组合： 1、#禁止修改； 2、#禁止添加； 4、#禁止删除； 8、#禁止自动判断（注：在默认情况下，系统将自动判断当前字节集字段的内容是否为图片或者声音，然后以对应的方式显示。使用本常量值可以禁止此功能）。 如果被省略，默认值为 0 。
字段范围 可选的； 可扩充的；文本型，参数数据可以同时提供数组或非数组数据。本参数的各参数值顺序指定所有欲编辑字段的名称。如果被省略，默认为所有字段。
