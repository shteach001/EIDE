纸张方向 整数型； 初始值为 0 。本成员设置打印纸的放置方向，可以为以下常量值之一： 0、#纵向； 1、#横向。
左边距 整数型； 初始值为 230 。本成员设置正文打印区到打印纸左边缘的距离，单位为0.1毫米。
上边距 整数型； 初始值为 230 。本成员设置正文打印区到打印纸上边缘的距离，单位为0.1毫米。
右边距 整数型； 初始值为 230 。本成员设置正文打印区到打印纸右边缘的距离，单位为0.1毫米。
下边距 整数型； 初始值为 230 。本成员设置正文打印区到打印纸下边缘的距离，单位为0.1毫米。
页号位置 整数型； 初始值为 0 。本成员设置是否打印页号及页号的打印位置。可以为以下常量值之一： 0、无页号；1、#上左页号； 2、#上中页号； 3、#上右页号； 4、#下左页号； 5、#下中页号； 6、#下右页号。
打印份数 整数型； 初始值为 1 。本成员设置打印时共打印的份数。
首页打印页号 整数型； 初始值为 1 。本成员设置打印时首页的打印页号。
是否打印到文件 逻辑型； 初始值为假。本成员设置是否将打印数据输出到指定文件。
打印文件名 文本型； 如果设置为打印到文件，本成员指定数据输出到的文件名。
自动填充 逻辑型； 初始值为假。本成员设置是否自动重复尾数据空行填充尾页空白。注意如果“每页打印行数”成员值大于零，本成员无效。
自动添加表格线 逻辑型； 初始值为假。本成员设置打印时是否自动为单元格添加边框线。
奇偶页方式 整数型； 初始值为 0 。本成员可为以下常量值之一： 0、#全部页； 1、#仅打印奇数页； 2、#仅打印偶数页。
打印范围方式 整数型； 初始值为 0 。本成员可为以下常量值之一： 0、#全部页； 1、#页范围； 2、#行范围。
起始打印位置 整数型； 初始值为 1 。如果打印范围方式为“页范围”，本成员记录首页页号（页号从1开始）；如果打印范围方式为“行范围”，本成员记录首行行号；如果打印范围方式为“全部页”，本成员无效。
结束打印位置 整数型； 初始值为 0 。如果打印范围方式为“页范围”，本成员记录末页页号（页号从1开始）；如果打印范围方式为“行范围”，本成员记录末行行号；如果打印范围方式为“全部页”，本成员无效。如果本成员值为 0 ，则将一直打印到所有数据行结束。
每页打印行数 整数型； 初始值为 0 。本成员设置每页打印行数(不包含表头行),如果为0则表示根据页高自动判别。
打印缩放比 整数型； 初始值为 0 。指定打印时所使用的缩放百分比，值为 -1、-2 或 20 到 1000 之间。其含义如下： -1：#缩放到纸宽。即自动压缩或扩张到纸张宽度打印； -2：#限定到纸宽。即：如果欲打印数据宽度超过纸张宽度，则自动压缩到纸张宽度，否则原样打印； 20 - 1000：直接指定打印缩放百分比； 如果指定为0，则默认为100。
